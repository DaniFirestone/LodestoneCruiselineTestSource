
# Blueprint: Lodestone Cruiseline Test Source

## Overview

This document outlines the structure, features, and ongoing development of the Lodestone Cruiseline Test Source, a web application built with Astro.js. The application serves as a demonstration of a searchable and filterable catalog of cruise line shore excursions.

## Project Outline

### Design and Styling

*   **Frameworks:** Astro.js with React for interactive components and Tailwind CSS for styling.
*   **Typography:** The UI uses a standard sans-serif font family.
*   **Color Palette:** The color scheme is based on a default Tailwind CSS configuration.
*   **Iconography:** The application uses a ship icon for the favicon and in the header.

### Features

*   **Excursion Catalog:** Displays a list of shore excursions from a static data source.
*   **Search and Filtering:** Users can search for excursions and filter them by various criteria.
*   **Detailed View:** A detailed view is available for each excursion, showing more information.
*   **Fleet Information:** The catalog displays information about the ships in the fleet.
*   **JSON Data View:** A button allows users to view the raw JSON data for the company and fleet information.
